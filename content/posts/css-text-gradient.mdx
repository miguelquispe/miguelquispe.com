En este tutorial, aprenderemos a incorporar un color degradado a nuestro texto mediante CSS. Con solo unos simples pasos mejoraremos el aspecto visual de nuestros proyectos web.

![Vista texto con degradado](/images/posts/css-text-gradient/preview.jpg)

## Paso 1: Agregar degradado

Primero, agregamos la propiedad `background` y luego utilizamos el valor `linear-gradient` con nuestros colores deseados dentro de nuestra clase `.title`.

En este ejemplo, empleo un degradado de tres colores, definiendo su direcci칩n de izquierda a derecha.

```css:style.css {3}
.title {
  font-size: 60px;
  background: linear-gradient(to right, #fa4261, #f39762, #d5aa10);
  background-size: cover;
}
```

![Vista texto con fondo degradado](/images/posts/css-text-gradient/bg-gradient.jpg)

## Paso 2: Enmascarar el degradado con el texto

Ahora emplearemos dos propiedades css: [`background-clip`](https://developer.mozilla.org/es/docs/Web/CSS/background-clip) y [`-webkit-text-fill-color`](https://developer.mozilla.org/es/docs/Web/CSS/-webkit-text-fill-color).

Con `background-clip: text` indicamos que el fondo se debe mostrar en el texto, y `-webkit-text-fill-color: transparent` removemos el color de relleno del texto.

```css:style.css {5,6}
.title {
  font-size: 60px;
  background: linear-gradient(to right, #fa4261, #f39762, #d5aa10);
  background-size: cover;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

De esta manera, ya hemos aplicado un degradado a nuestro texto.

![Vista texto con degradado](/images/posts/css-text-gradient/text-gradient.jpg)

## Plus: Agregar fuente (font) personalizada

En este ejemplo, voy a utilizar la fuente [Great Vibes](https://fonts.google.com/specimen/Great+Vibes) de Google Fonts.

Para lograrlo, a침adimos la etiqueta `<link />` para vincular con Google Font, todo dentro del elemento `head` de nuestro archivo `index.html`.

```html:index.html {5-8}

  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link
    href="https://fonts.googleapis.com/css2?family=Great+Vibes&display=swap"

```

Luego, asignamos la fuente **Great Vibes** como `font-family` de nuestro texto.

```css:style.css {2,3}

  font-family: 'Great Vibes', cursive;
  font-size: 86px;
  /* ... */
}
```

```

## Soporte para navegadores

Si quieres asegurarte que el efecto se muestre solo en navegadores que soporten esta caracter칤stica, puedes usar la regla [`@supports`](https://developer.mozilla.org/es/docs/Web/CSS/@supports).

`@supports` permite aplicar estilos solo si el navegador soporta una caracter칤stica en espec칤fico.

Adem치s, podemos usar un color s칩lido como **fallback** (alternativa) para navegadores que no soporten el efecto.

  font-size: 60px;
  color: #fa4261;
}

  (-webkit-background-clip: text) or (-webkit-text-fill-color: transparent)

    -webkit-text-fill-color: transparent;
  }
}
```
